# Data Structures in C

This repository contains C language implementations developed during the **Data Structures** course.

The main goal is to document the learning process through practical examples, focusing on fundamental programming concepts and the development of problem-solving skills.

---

## Contents

The repository includes implementations of:

- Arrays and matrices
- Pointers
- Basic algorithms
- Sorting methods
- Other foundational data structures (as the course progresses)

Each folder represents a specific topic or exercise proposed during the classes.

---

## Purpose

This project was created to:

- Practice C programming
- Understand core data structures
- Improve logical thinking
- Build a solid foundation for more advanced topics in computer science

---

## Technologies

- Language: **C**
- Tools: GCC / VS Code
